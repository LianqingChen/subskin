## 第二阶段：AI病情量化评估工具开发 (Day 4-10) ⭐最高优先级

### 核心功能需求

**用户流程**:
1. 访问"AI病情量化追踪"页面
2. 手机号验证码快速登录（首次使用）
3. 上传白斑清晰照片
4. 系统自动识别白斑面积、分型，输出VASI评分
5. 生成量化评估报告
6. 用户保存评估记录
7. 查看历史变化曲线，对比治疗效果

**公开承诺**: 该工具永久免费，无使用次数限制

## 附录：技术栈确认

### 前端
- **框架**: VitePress (Vue 3)
- **图表库**: Chart.js 或 ECharts
- **状态管理**: Pinia (如需要)

### 后端
- **API框架**: FastAPI
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **认证**: JWT + 手机号验证码
- **图像存储**: 对象存储（阿里云OSS / 腾讯云COS）

### 第三方服务
- **VASI识别API**: 待选（百度AI / 腾讯云 / 第三方医疗AI）
- **短信服务**: 阿里云短信 / 腾讯云短信
- **LLM服务**: 已有（OpenAI / Volcengine）

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|---------|

| 第三方服务不稳定 | 中 | 中 | 实现重试机制和缓存策略 |
