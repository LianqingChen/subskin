## 💾 数据存储

### 原始数据
- 位置: `/root/subskin/data/raw/`
- 命名格式: `{source}_{YYYY-MM-DD}.json`
- 格式: JSON 数组，每条包含完整元数据

### 增量数据库
- 位置: `/root/subskin/incremental_updates.sqlite`
- 表: `daily_updates`
- 每条记录包含:
  - `id`: 唯一ID
  - `update_type`: 新增/更新分类
  - `resource_id`: 资源ID (PMID / NCT编号 / URL)
  - `resource_title`: 标题
  - `change_details`: JSON 详细信息（包含原文链接）
  - `timestamp`: 收集时间
  - `source`: 信息来源分类
  - `processed`: 是否已AI总结
