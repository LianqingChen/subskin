# 心理调节与情绪支持

> 得了白癜风，心理压力往往比身体痛苦更大。这里聊聊怎么调整心态，和白癜风和平共处。

## 你不是一个人

很多刚确诊的朋友都会经历这几个阶段：
1. **否认**：不可能！我怎么会得这个病？
2. **愤怒**：为什么是我？我做错了什么？
3. **抑郁**：我好绝望，人生完蛋了
4. **接受**：虽然它不好，但我还是能好好生活

请记住：**这些情绪都是正常的**。

## 调节建议

### 1. 接受它

白癜风不影响寿命，不会要你命，只是影响美观。接受它的存在，带着它一起好好生活。

### 2. 找朋友聊聊

不要把情绪都藏在心里，来[心情驿站](../chat/mood.md)说一说，大家都会给你加油。

### 3. 专注于能控制的事情

- 你能控制规律作息
- 你能控制坚持治疗
- 你能控制调整心态

接受你不能控制的，改变你能改变的。

### 4. 培养一个爱好

把注意力从白斑上移开，去做自己喜欢的事情，让生活充实起来。

## 病友经验分享

> "刚开始确诊在脸上，我不敢出门，不敢见人，躲在家里哭了一个星期。现在两年过去了，虽然还没有完全好，但是控制住了，我也敢正常上班交朋友了。记住一句话：坚持就有希望。"
