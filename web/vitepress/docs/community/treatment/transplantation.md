# 移植治疗经验

> 表皮移植、黑色素细胞移植等手术治疗经验分享。

## 哪些情况适合移植

- 稳定期白癜风
- 白斑面积不大
- 药物和光疗效果不好
- 不在容易扩散的部位

## 常见手术方式

### 表皮移植

- 原理：取自己正常部位的表皮，移植到白斑部位
- 优点：技术成熟，费用相对不高
- 缺点：供皮区需要取皮，面积有限制
- 恢复时间：大概1-2周不能沾水

### 黑色素细胞移植

- 原理：培养黑色素细胞后移植
- 优点：可以处理较大面积
- 缺点：费用更高，技术要求高
- 经验分享：（待补充病友经验）

## 手术前后注意事项

- **术前**：确保白斑稳定至少半年以上
- **术后**：一定要遵医嘱护理，避免活动移植部位
- **成功率**：不是100%成功，部分可能需要二次手术
